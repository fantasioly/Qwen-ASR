  1. Change `clean_text, lang, _ = await task` to `clean_text, lang, usage = await task`
  2. Build the send_json payload conditionally including usage:
     ```python
     payload = {"type": "partial", "text": clean_text, "language": lang}
     ```

  **Part B — Frontend (RealTimePanel.tsx):**

  The hook's `finalUsage` may be null when the final transcription times out, but the user already heard partial results. The cache badge and stats row should render whenever `finalUsage` is available OR when a partial frame carried usage.

  However, `useStreamingTranscribe` does not currently persist partial usage data. The `StreamingClient.onPartial` callback only receives text. We need to thread usage through:

  1. In `frontend/src/api/streaming.ts` (onPartial at line 203-204): the `frame.usage` is already typed on `StreamingResponse`. But `onPartial` signature is `(text: string) => void`. Change it to `(text: string, usage?: StreamingResponse['usage']) => void` and pass `frame.usage` as second arg.

  2. In `frontend/src/hooks/useStreamingTranscribe.ts`:
     - Add `partialUsage` state: `useState<TranscribeUsage | null>(null)`
     - In the `onPartial` callback, update: `(text, usage) => { setPartialText(text); if (usage) setPartialUsage(usage); }`
     - Expose `partialUsage` in the return value

  3. In `RealTimePanel.tsx`:
     - Destructure new `partialUsage` from hook
     - Resolve usage: `const displayUsage = finalUsage ?? partialUsage`
     - Replace all `finalUsage &&` guards with `displayUsage &&`
     - Replace `finalUsage.xxx` references with `displayUsage.xxx` in cache badge (line 64-76) and stats row (line 90-101)

  Do NOT change the `useStreamingTranscribe` TypeScript interface file — it's co-located in the same file. Modify the interface and the implementation in `useStreamingTranscribe.ts` together.

     ```tsx
     />
     ```

     ```tsx
     const handleQueueRemove = useCallback(

         const fullIndex = jobs.indexOf(job);
         if (fullIndex !== -1) removeJob(fullIndex);
       },
       [activeJobs, jobs, removeJob],
     );
     ```
     Then pass `onRemove={handleQueueRemove}`.
